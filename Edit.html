TODO Bearbeiten

Editieren eines TODOs

Ein neues TODO anlegen können sie auf der Seite TODO anlegen, indem Sie die nötigen Daten im ensprechenden Format eingeben und anschließend "Erstellen" klicken. Sollten Sie ein bereits bestehendes TODO verändern wollen, um beispielsweise den Fortschritt anzupassen, gehen Sie bitte auf unsere Übersichsseite und klicken dort bei dem jeweiligen TODO auf "Bearbeiten" beziehungsweise "Löschen".
